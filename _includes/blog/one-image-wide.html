{% comment %} Este módulo tiene como finalidad mostrar una imagen dentro de una entrada de blog, formateándola mediante Boostrap y ocupando todo el ancho de página estándar establecido por Bootstrap. Para que funcione, se tienen que asignar estas variables en Liquid: {% assign img_url = "/assets/img/components/myimage.jpg" %} {% assign img_description = "My image description" %} {% assign img_style = "max-height: 200px;" %} Esta última es opcional y permite asignar estilos a la imagen. Por ejemplo, limitar su altura o darle cualquier otra característica. Y luego hacer un include de esta forma: {% include blog/one-image.html %} {% endcomment %}
