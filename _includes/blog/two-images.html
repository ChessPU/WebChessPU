{% comment %} Este módulo tiene como finalidad mostrar dos imágenes dentro de una entrada de blog, formateándola mediante Boostrap. Las imágenes estarán en la misma línea si la pantalla es grande o una debajo de otra si la pantalla es más pequeña. Para que funcione, se tienen que asignar estas variables en Liquid: {% assign img1_url = "/assets/img/misc/kaos-3.jpeg" %} {% assign img1_description = "Las conexiones a la placas de interfaz se sueltan" %} {% assign img1_style = "border: 2px solid silver;" %} {% assign img2_url = "/assets/img/misc/kaos-4.jpeg" %} {% assign img2_description = "Los conectores se desprenden" %} {% assign img2_style = "border: 2px solid silver;" %} img1_style e img2_style son opcionales y permiten asignar estilos a las imágenes. Por ejemplo, limitar su altura o darle cualquier otra característica. Y luego hacer un include de esta forma: {% include blog/two-images.html %} {% endcomment %}
